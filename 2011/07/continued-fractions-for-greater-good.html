Continued Fractions for the Greater Good part 1

Jul 10, 2011

OK, maybe not for the greater good, but still fun. This first post will be relatively short and sweet, intended to give an introduction for the posts that will follow.

Before the introduction, some motivation courtesy of Wikipedia:

...decimal representation has some problems. One problem is that many rational numbers lack finite representations in this system. For example, the number 13\frac{1}{3}31​ is represented by the infinite sequence (0,3,3,3,3,…)(0, 3, 3, 3, 3, \ldots )(0,3,3,3,3,…). Another problem is that the constant 101010 is an essentially arbitrary choice, and one which biases the resulting representation toward numbers that have some relation to the integer 101010. For example, 1371600\frac{137}{1600}1600137​ has a finite decimal representation, while 13\frac{1}{3}31​ does not, not because 1371600\frac{137}{1600}1600137​ is simpler than 13\frac{1}{3}31​, but because 160016001600 happens to divide a power of 101010 (106=1600×625)(10^6 = 1600 \times 625)(106=1600×625). Continued fraction notation is a representation of the real numbers that avoids both these problems. Let us consider how we might describe a number like 41593\frac{415}{93}93415​, which is around 4.46244.46244.4624. This is approximately 444. Actually it is a little bit more than 444, about 4+124 + \frac{1}{2}4+21​. But the 222 in the denominator is not correct; the correct denominator is a little bit more than 222 about 2+162 + \frac{1}{6}2+61​, so 41593\frac{415}{93}93415​ is approximately

4+12+16.4 + \cfrac{1}{2+\cfrac{1}{6}}.4+2+61​1​.
But the 666 in the denominator is not correct; the correct denominator is a little bit more than 666, actually 6+176+\frac{1}{7}6+71​. So 41593\frac{415}{93}93415​ is actually
4+12+16+17.4 + \cfrac{1}{2+\cfrac{1}{6 +\cfrac{1}{7}}}.4+2+6+71​1​1​.
This is exact...

With this in mind, one can define an infinite continued fraction to be

a0+1a1+1a2+⋱.a_0 + \cfrac{1}{a_1 +\cfrac{1}{a_2 +\ddots}}.a0​+a1​+a2​+⋱1​1​.

With the denominators a0,a1,a2,…a_0, a_1, a_2, \ldotsa0​,a1​,a2​,…, we can define a recurrence for the finite approximations (convergents) of this value. For example, the zeroth is a0a_0a0​ and the first is

a0+1a1=a0a1+1a1.\displaystyle a_0 + \frac{1}{a_1} = \frac{a_0 a_1 + 1}{a_1}.a0​+a1​1​=a1​a0​a1​+1​.

The other motivation (the one I actually learned first in real life) for continued fractions comes from 2\sqrt{2}2​ being represented by an infinite continued fraction. (Instead of saying a probability of 0.018760.018760.01876, people would rather say a 111 in 535353 chance.) So we try to write 2=1.41421356…\sqrt{2} = 1.41421356\ldots2​=1.41421356… as 1+12.4141 + \frac{1}{2.414}1+2.4141​. But, instead, notice that

2=1+(2−1)=1+12+1.\sqrt{2} = 1 + (\sqrt{2} - 1) = 1 + \frac{1}{\sqrt{2} + 1}.2​=1+(2​−1)=1+2​+11​.

Plugging this into itself, we have

2=1+11+1+12+1=1+11+1+11+1+12+1\sqrt{2} = 1 + \cfrac{1}{1 +1 + \cfrac{1}{\sqrt{2} + 1}} =1 + \cfrac{1}{1 +1 + \cfrac{1}{1 + 1 + \cfrac{1}{\sqrt{2} + 1}}}2​=1+1+1+2​+11​1​=1+1+1+1+1+2​+11​1​1​

and notice it can be represented by (1;2,2,2,…)(1; 2, 2, 2, \ldots)(1;2,2,2,…).

Define the nnnth convergent to be hnkn\frac{h_n}{k_n}kn​hn​​, so above we have h0=a0,k0=1h_0 = a_0, k_0 = 1h0​=a0​,k0​=1 and h1=a0a1+1,k0=a1h_1 = a_0 a_1 + 1, k_0 = a_1h1​=a0​a1​+1,k0​=a1​.

Claim:

hnh_nhn​ and knk_nkn​ satisfy

hn=anhn−1+hn−2kn=ankn−1+kn−2\begin{aligned}h_n &= a_n h_{n - 1} + h_{n - 2} \\k_n &= a_n k_{n - 1} + k_{n - 2}\end{aligned}hn​kn​​=an​hn−1​+hn−2​=an​kn−1​+kn−2​​

along with h−1=1,h−2=0h_{-1} = 1, h_{-2} = 0h−1​=1,h−2​=0 and k−1=0,k−2=1k_{-1} = 0, k_{-2} = 1k−1​=0,k−2​=1.

Proof:

The fraction hnkn\displaystyle \frac{h_n}{k_n}kn​hn​​ is converted into hn+1kn+1\displaystyle \frac{h_{n + 1}}{k_{n + 1}}kn+1​hn+1​​ simply by changing ana_nan​ to an+1an+1a_n + \frac{1}{a_{n + 1}}an​+an+1​1​ in the final denominator. Since

hnkn=anhn−1+hn−2ankn−1+kn−2\frac{h_n}{k_n} = \frac{a_n h_{n - 1} + h_{n - 2}}{a_n k_{n - 1} + k_{n - 2}}kn​hn​​=an​kn−1​+kn−2​an​hn−1​+hn−2​​

we similarly have

hn+1kn+1=(an+1an+1)hn−1+hn−2(an+1an+1)kn−1+kn−2=an+1(anhn−1+hn−2)+hn−1an+1(ankn−1+kn−2)+kn−1=an+1hn+hn−1an+1kn+kn−1\begin{aligned}\frac{h_{n + 1}}{k_{n + 1}} &= \frac{\left(a_n + \frac{1}{a_{n + 1}}\right)h_{n - 1} + h_{n - 2}}{\left(a_n + \frac{1}{a_{n + 1}}\right)k_{n - 1} + k_{n - 2}} \\ &= \frac{a_{n + 1}(a_n h_{n - 1} + h_{n - 2}) + h_{n - 1}}{a_{n + 1}(a_n k_{n - 1} + k_{n - 2}) + k_{n - 1}} \\&= \frac{a_{n + 1} h_n + h_{n - 1}}{a_{n + 1} k_n + k_{n - 1}}\end{aligned}kn+1​hn+1​​​=(an​+an+1​1​)kn−1​+kn−2​(an​+an+1​1​)hn−1​+hn−2​​=an+1​(an​kn−1​+kn−2​)+kn−1​an+1​(an​hn−1​+hn−2​)+hn−1​​=an+1​kn​+kn−1​an+1​hn​+hn−1​​​

Thus hn+1h_{n + 1}hn+1​ and kn+1k_{n + 1}kn+1​ satisfy the same recurrence.

It remains to check the initial conditions work, but note

h0=a0h−1+h−2=a0⋅1+0=a0k0=a0k−1+k−2=a0⋅0+1=1\begin{aligned}h_0 &= a_0 h_{-1} + h_{-2} = a_0 \cdot 1 + 0 = a_0 \\k_0 &= a_0 k_{-1} + k_{-2} = a_0 \cdot 0 + 1 = 1 \end{aligned}h0​k0​​=a0​h−1​+h−2​=a0​⋅1+0=a0​=a0​k−1​+k−2​=a0​⋅0+1=1​

and

h1=a1h0+h−1=a0a1+1k1=a1k0+k−1=a1⋅1+0=a1\begin{aligned}h_1 &= a_1 h_{0} + h_{-1} = a_0 a_1 + 1 \\k_1 &= a_1 k_{0} + k_{-1} = a_1 \cdot 1 + 0 = a_1 \end{aligned}h1​k1​​=a1​h0​+h−1​=a0​a1​+1=a1​k0​+k−1​=a1​⋅1+0=a1​​

as we checked above. ■\blacksquare■
